# Mini Test Suite for Skatteverket's Personal Number API

## Requirements
- JDK 17

## Setup
Run this Maven Wrapper script. It downloads and runs Maven version 3.9.9, and runs all tests.
```bash
./mvnw test      # Unix/macOS
.\mvnw.cmd test  # Windows
```
